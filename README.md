## Install Laravel

    laravel new Sanctum-Api

## Install Sanctum

    php artisan install:api

## Make Controller

    php artisan make:controller Api/AuthController


## API Route

    Signup : http://localhost:8000/api/signup
    Login  : http://localhost:8000/api/login

## Logout 

    http://localhost:8000/api/logout
